# Adapters

This directory contains configurations that connect the content of this library
to AI coding tools via [Ruler](https://github.com/intellectronica/ruler).

## `ruler/`

A single adapter that works with **every** ruler-supported agent at once:

- [`ruler/README.md`](ruler/README.md) — setup guide (rules + skills + workflows)
- [`ruler/ruler.toml`](ruler/ruler.toml) — ruler config with all common agents enabled

See [`ruler/README.md`](ruler/README.md) for full instructions.
