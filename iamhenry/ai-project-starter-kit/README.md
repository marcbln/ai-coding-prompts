# Project Starter Kit

A comprehensive starting point for software development projects leveraging **Roo Code** AI-assisted code generation and structured planning.

## Overview

This starter kit provides a foundation for new software projects with a focus on organized documentation and AI-assisted development using Roo Code. It includes templates and documentation structures designed to streamline the development process from planning to implementation.

## Features

- **Roo Code Optimized**: Pre-defined templates with frontmatter metadata for Roo Code's custom tool system
- **AI-Assisted Code Generation**: XML-tagged prompts for TDD, code review, planning, and quality assurance
- **Structured Documentation**: Frameworks for planning and documenting your software projects
- **Development Workflow Guides**: Best practices for organized development

## Getting Started

1. Clone this repository as a base for your new project
2. Explore the documentation in the `docs` folder for templates and guides
3. Adapt the structure to fit your specific project needs

## Documentation

The `_ai/tools/` directory contains Roo Code tool definitions organized by category:
- **TDD Tools**: Test-driven development workflows (red/green/refactor phases)
- **Planning**: OODA, SAPPO, and reasoning frameworks
- **Quality**: Code guidelines and hybrid code review process
- **Generators**: Changelog, memory, task generation, and ASO

## Purpose

This starter kit is designed to:
- Accelerate project setup and planning
- Ensure consistent documentation
- Optimize interaction with Roo Code AI assistant
- Provide structure for complex software projects
