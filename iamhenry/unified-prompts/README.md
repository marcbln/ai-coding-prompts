# Unified Prompts

This directory contains a merged hierarchy of prompts from multiple sources:
- `modes/` - AI coding mode definitions
- `fetched-prompts/` - Collected prompt templates
- `ai-project-starter-kit/_ai/tools/` - Development workflow prompts

## Directory Structure

```
unified-prompts/
├── code-analysis/          # Code analysis and review prompts
├── code-design/             # Architecture and design patterns
├── code-development/       # Implementation, TDD, generators
│   └── tdd/                # Test-driven development workflow
├── code-documentation/     # Documentation generation
├── code-orchestration/     # Workflow coordination
├── code-quality/           # Quality assurance and review
├── consulting/             # Consulting and advisory prompts
├── design/                 # Design frameworks
├── documentation/          # Documentation templates
├── marketing/              # Marketing frameworks and strategies
├── product/                # Product management prompts
└── workflow/               # Workflow and process templates
```

## Usage

Browse each category to find relevant prompts for your specific task.
